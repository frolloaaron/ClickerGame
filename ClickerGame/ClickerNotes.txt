Cake Clicker

Upgrades:
- Various upgrades to improve output
- Progression should be fast enough to continuously make progress but not too fast to be mindless

--- GAME FEATURES ---
- Main Game -
-Big cake button to press to make number go up
-Display of current cake count
-Store with upgrades to increase cake count

- Stretch Goals -
-Can press button with multiple keys
-Can Buy multiple at once

--- GAME STRUCTURE ---
How are we handling display count?
- Store amount of cakes in a double variable, set one timer, convert to integer when displayed

How are we handling upgrades (buying + math)?
- Create an Upgrade class
	- Cakes/Second
	- Current Cost
	- Name
	- Picture
	- How many of the upgrade the user currently has
